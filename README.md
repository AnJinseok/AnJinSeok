# AnJinSeok

PG(Payment Gateway) 업종에서 일하고 있으며, **백엔드 · API · 데이터 파이프라인** 역량을 보여주기 위한 사이드 프로젝트를 운영하고 있습니다.

---

## 주요 프로젝트: Maple API & Front

게임 데이터 파이프라인 (스크래핑 → API → 웹 UI)

| 단계         | 기술                  |
| ------------ | --------------------- |
| 데이터 수집  | Python (스크래핑)     |
| 백엔드       | Spring Boot, MariaDB  |
| 프론트엔드   | React, Vite           |

**체험** · [maple85.ddns.net](https://maple85.ddns.net) (HTTPS)

**저장소** · [maple-api](https://github.com/AnJinseok/maple-api) (REST API) · [maple-front](https://github.com/AnJinseok/maple-front) (웹 UI)
